Aztec

Datamost, 1982 — Paul Stephenson — action-adventure in the tomb of Quetzalcoatl

Explore a randomly generated Aztec pyramid to recover a jade idol and escape alive. Fight snakes, spiders, and Aztec guards with a machete and pistol. Blow through walls with dynamite. Navigate traps, pits, and collapsing staircases. The dungeon layout changes every game.

Getting started

  1. The game boots from disk. Select a difficulty level (1–8).
  2. You start at the entrance with a machete, pistol, and dynamite.
  3. Descend through the pyramid, find the jade idol, and return to the entrance.
  4. Press Esc at any time for the in-game help screen.

Strategy

  • Press Esc for help. The in-game help screen lists all commands.
  • Save dynamite. It's rare — use it to blow through walls or floors to create shortcuts.
  • Crawl through low passages. Press G to get through tight gaps.
  • Conserve pistol ammo. Use the machete for weak enemies; save bullets for guards.
  • Don't take too long. The idol's value decreases over time — speed is rewarded.
  • Remember the way out. You must return to the entrance after finding the idol.
