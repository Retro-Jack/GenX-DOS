15 Puzzle

Jeff Jetton, 2020 — 6502 assembly sliding puzzle for the Apple I

Arrange 15 letter tiles (A–O) in alphabetical order on a 4×4 grid by sliding them into the empty space. Choose a difficulty level that determines how thoroughly the board is shuffled, then solve it in as few moves as possible.

Getting started

  1. The game loads via turbo tape and asks for a difficulty level (1–5).
  2. Level 1 shuffles lightly (3 moves); level 5 shuffles heavily (255 moves).
  3. The scrambled board is displayed. Start sliding tiles.

Controls

KeyAction
A–OSlide that letter tile towards the empty space. The tile must be on the same row or column as the empty space. Any tiles between it and the space slide too.
QQuit the game

Goal

Arrange the tiles so the board reads:

  A B C D
  E F G H
  I J K L
  M N O _

The game tracks your move count and shows a celebratory message when you solve it.

Strategy

  • Work top-down. Get the top row correct (A B C D) first, then the second row (E F G H), then solve the bottom two rows together.
  • Lock rows in place. Once a row is correct, avoid disturbing it while working on lower rows.
  • Corner technique. To place the last two tiles of a row, position them as a column first, then slide them into place together.
  • Start on level 2 or 3. Level 1 is trivially easy; level 5 can be very challenging.

Apple I keyboard

Single letter keypresses — no Return needed.
