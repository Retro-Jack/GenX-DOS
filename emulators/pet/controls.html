Commodore · PET 4032

Commodore PET — Controls

VICE xpet (gold-standard Commodore PET emulator) running in your browser via EmulatorJS, configured for the PET 4032 — 32 KB RAM, BASIC 4.0, 40-column green-on-black monochrome CRT. The PET 2001 was Commodore's first computer (Jan 1977) and effectively founded the home-computer market; the 4032 (1980) added BASIC 4 and a proper keyboard. Click the canvas first so the emulator receives keyboard input.

Most bundled games are single-file BASIC .prgs that load instantly via VICE autostart. The PETSCII Robots shareware is a .d64 disk image — that one takes ~30 seconds to boot through real-time IEC drive emulation (you'll hear the drive chunk the whole way), then drops into the title screen.

Keyboard-only input

The PET has no joystick port. All games are keyboard-controlled. PETSCII Robots uses:

KeyAction
W A S DMove (up / left / down / right)
SpaceAction / fire / pick up
IInventory
PPause

Most older PET games map directional input to keys near the right-hand side of the keyboard (; ' for left/right is common), or use number keys. Read the on-screen instructions when each game boots.

PET keyboard

VICE maps the PET keyboard "positionally" to your PC keyboard — letters / numbers type as themselves. PET-specific keys:

KeyPET key
Scroll LockRUN/STOP — breaks out of BASIC programs (remapped from Esc — see note below)
L AltCommodore-style graphic / inverse toggle
Page UpRVS ON (reverse video)
Page DownRVS OFF
HomeCLR/HOME
BackspaceINST/DEL
F1 – F10Function keys (model-dependent — most 4032 software ignores)

Why Scroll Lock for RUN/STOP? Browsers capture Esc to exit pointer-lock and fullscreen before the libretro keyboard callback can see it, so a real Esc press never reaches VICE. The bundle remaps Scroll Lock via emulators/_shared/genx-vice-softkeys.js — a document-level keydown listener swallows the Scroll Lock press and dispatches a synthetic Escape event (synthetic events don't trigger the browser's Esc-capture since they aren't real user gestures), which then reaches the core as RUN/STOP. Scroll Lock has no other use in this bundle so the remap is free.

Loading from BASIC 4

VICE auto-LOADs .prg files and the first program on most disks. If you drop to the READY. prompt, type one of:

CommandEffect
DIRECTORYList the disk directory (BASIC 4 shortcut — equivalent to LOAD"$",8)
DLOAD"name"Load a program from disk (BASIC 4 shortcut)
RUNRun the loaded program

About PETSCII Robots

PETSCII Robots is a modern (2022) survival/strategy title by David Murray ("The 8-Bit Guy") — proof that the 32K PET still has commercial-grade games being written for it 45 years after launch. This is the official shareware build (free demo); the full game is sold by Murray directly. The shareware includes the first three of the full game's 30+ levels.

Emulator: VICE xpet (GPL-2.0, libretro core) under EmulatorJS (GPL-3.0). Both shipped locally from this site; no network calls at runtime.
