Commodore 64 — Controls

A Commodore 64. Most C64 games use the joystick on port 2, which is mapped to the keyboard. ROMs are loaded via vc64web's own settings page (drag-and-drop, persists in your browser).

Joystick (port 2 — Player 1)

KeyAction
↑ ↓ ← →Stick direction
L Ctrl or SpaceFire button

If a game ignores Player 1 input, it's looking at port 1 instead — vc64web's bottom toolbar has a port-swap button, or the per-game config in this site flags port1: true where known.

Loading from BASIC

If a disk drops to the READY. prompt, type one of:

CommandEffect
LOAD"$",8List the directory of disk drive 8
LOAD"*",8,1Load + run the first program on the disk
LOAD"FILENAME",8,1Load a specific PRG and place it where the file expects
RUNRun the loaded program

vc64web UI

vc64web has a bottom toolbar (hover the bottom edge): pause / reset / fullscreen / port swap / virtual joystick / virtual keyboard / settings. Right-click the canvas for additional menu options.

First-time ROM setup

vc64web needs four ROM files (KERNAL, BASIC, CHARGEN, 1541 disk drive) to run. On first visit it opens its ROMs settings page — drag your four ROM dumps onto the matching drop targets. The browser remembers them in localStorage, so this is a one-time step per browser.

Notes

Most bundled titles are .d64 disk images. With the disk drive ROM loaded, vc64web auto-runs them (the embed config sets auto load and auto run).
