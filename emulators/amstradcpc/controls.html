Amstrad · CPC 6128

Amstrad CPC — Controls

Caprice32 (cap32 libretro) running in your browser via EmulatorJS, configured as the CPC 6128 (128 KB RAM, BASIC 1.1, 3-inch floppy drive). Released 1984 by Amstrad as the rival to the C64 and Spectrum in the UK / Europe / France — the CPC range sold ~3 million units and built one of the strongest 8-bit game libraries outside Japan. Click the canvas first so the emulator receives keyboard input.

Joystick (mapped to keyboard)

Most CPC games support joystick input on the dedicated 9-pin port. Caprice32 maps this to the keyboard automatically:

KeyAction
↑ ↓ ← →Joystick directions
XFire 1
ZFire 2
SpaceOften "start" or "pause"; varies by game

CPC keyboard (for games that need typing)

The CPC keyboard is mapped positionally: letters / digits type as themselves. CPC-specific keys:

KeyCPC key
EnterRETURN
EscESC (often "pause" or "quit" in-game)
BackspaceDEL
TabTAB

Loading from disk

cap32 auto-mounts the bundled `.dsk` to drive A:. Some games autoboot via the menu, others land at the BASIC `Ready` prompt. To start manually:

CommandEffect
catList the disk's directory
run"disc"Run the first game on most disks (CPC menu disc convention)
run"name"Run a specific program by filename (use what `cat` shows)
