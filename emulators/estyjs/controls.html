Notes

Many cracked ST disk releases (the bundled M.U.L.E.-era titles in particular) show a brief intro / cracker greet screen before the game starts — wait it out or press fire.

Heavily protected ST games (e.g. Captain Blood) use the Pasti .stx format, which EstyJS doesn't support. Only .st / .msa images work here.

This build doesn't expose a runtime menu; to load a different disk, use the back button + a different menu entry.

Mapping follows EstyJS 2.0 (GPL-2.0+, Kai Eckert / Darren Coles). EmuTOS is bundled (GPL-2.0+, open-source TOS reimplementation — no copyrighted Atari TOS).
