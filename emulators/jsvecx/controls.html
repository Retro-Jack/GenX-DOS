GCE · Vectrex (1982)

Vectrex — Controls

The Vectrex shipped with one analogue joystick that had four action buttons in a row. PC keyboard maps the joystick directions and the four buttons.

↑ ↓ ← →Stick direction (analogue motion is faked digitally)

Action buttons (1 / 2 / 3 / 4)

The Vectrex controller's four buttons sat in a horizontal row labelled 1, 2, 3, 4. JSVecX maps them to the home-row + space:

KeyVectrex button
AButton 1 (typically fire / accelerate)
SButton 2
DButton 3
FButton 4

Notes

The Vectrex's BIOS plays the "Vectrex" intro tune at every boot — wait it out (about 4 seconds) before pressing buttons.

Most cartridges have no in-game pause; use the toolbar Pause button.

Mapping follows JSVecX (GPL-3.0, DrSnuggles fork of raz0red's port of Valavan Manohararajah's VecX). The on-screen "controls" image inside the bundle's img/controls.png shows the same mapping with a controller diagram.
