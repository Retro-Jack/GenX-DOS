Sinclair Spectrum — Controls

A 48K Spectrum with its rubber-key keyboard. The bundled titles are .z80 snapshots — they boot mid-game, so most controls live in the game itself.

Spectrum keyboard

Most letter / number / symbol keys map directly. The Spectrum's shift behaviour is split into two keys:

KeySpectrum key
ShiftCAPS SHIFT
CtrlSYMBOL SHIFT
EnterENTER
BackspaceCAPS SHIFT + 0 (DELETE)

Kempston joystick (Player 1)

JSSpeccy 3 emulates the Kempston joystick interface on the keyboard. Most action games default to Kempston:

KeyAction
↑ ↓ ← →Stick direction
AltFire button

If a game uses Sinclair Interface 2 instead (rare in the bundled titles), use 6 7 8 9 + 0 for Player 1.

UI

JSSpeccy 3 shows a central play overlay on first load — click it to start emulation (browsers require a user gesture before Web Audio can start). After that, the canvas drives input directly.

Mapping follows JSSpeccy 3.2 (GPL-3.0, gasman/jsspeccy3). The full reference (including the on-screen keyboard for touch devices) lives in the upstream README.
