MS-DOS · DOSBox (js-dos)

MS-DOS — Controls

js-dos runs DOSBox in WebAssembly. Each game has its own keys — there's no global mapping. The notes below cover the per-game conventions for the bundled titles plus the js-dos host shortcuts.

Common DOS-game conventions

Game familyTypical keys
Wolfenstein 3D, DoomArrows = move, Ctrl = fire, Space = open / use, 1–7 = weapon, Esc = menu
Duke Nukem 1 / 2Arrows = move, Ctrl = jump, Alt = fire, Esc = menu
Duke Nukem 3DArrows = move, A / Z = look, Ctrl = fire, Space = open, Tab = map
Commander Keen 1–6Arrows = move, Ctrl = jump, Alt = fire pogo / shoot, Esc = menu
Prince of PersiaArrows = move, Shift = grab / draw sword, Up+Shift = stand still, Esc = menu, Ctrl+A = restart level
SimCityMouse-driven (click + drag); F1–F12 for tool select
Tyrian, Epic PinballArrows + Space/Ctrl; in-game options menu rebinds
Out of This World, ZorkMostly keyboard text + arrows; check the in-game help (F1 in most adventures)

Notes

Audio doesn't start until you click the canvas — Web Audio autoplay policy. Most DOSBox configs auto-mount C: with the bundled game files; if a game needs Y/N at the install prompt, the bundle has it pre-installed.

Shareware episodes (Duke 1, Wolf3D, Doom, Keen, Tyrian) start at the first level / episode without nag screens. Purchase prompts in the bundled shareware versions can be bypassed with Esc.

Host emulator: js-dos v7.0.0 (GPL-2.0, caiiiycuk/js-dos), wrapping a DOSBox build (GPL-2.0). Per-game key mappings are defined inside each .jsdos bundle's DOSBox config.
