Acorn Archimedes — Controls

An Acorn A3000 / A3010-class machine running RISC OS 3. Three-button mouse drives the desktop; keyboard is a standard Acorn 102-key layout.

Game keys

Most Archimedes games use the keyboard rather than a joystick (the platform shipped without one). Common conventions:

KeysTypical use
Z XLeft / Right
' / or Shift + ↑ / ↓Up / Down
Return or SpaceFire / action

Emulator UI

The bundled archimedes-live shell has a top toolbar (hover the top edge): change disc, change machine model (A3000 / A3010 / A3020), reset, full-screen.

Notes

RISC OS games often need to be launched from the disc filer — double-click the disc icon at the bottom of the desktop, then double-click the application's directory.

Per-title ff-ms fast-forward is honoured automatically so games skip the RISC OS boot delay.

Mapping follows pdjstone/archimedes-live (MIT, WASM port of Sarah Walker's Arculator). The desktop tour at https://archi.medes.live/ covers the upstream UI in more detail.
