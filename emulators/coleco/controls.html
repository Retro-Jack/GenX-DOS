ColecoVision — Controls

Self-hosted gearcoleco (libretro Z80 + TMS9918A core) running in your browser via EmulatorJS. ColecoVision launched in August 1982 with Donkey Kong as the pack-in title and out-sold the Atari 2600 in its first year — the first console with near-arcade-quality graphics at home. The system shipped with a numeric-keypad controller and bundled arcade-overlay cards for each game. Click the canvas first so the emulator receives keyboard input.

Numeric keypad — overlay buttons

The ColecoVision controller had a 12-key telephone-style keypad (1–9, *, 0, #) — a numpad in everything but name. The site drives the whole keypad off your PC's numeric keypad. NumLock must be ON or the keys read as arrows/home/etc. — a banner fires if it's off.

KeyKeypad button
Numpad 1 – Numpad 8Keypad 1–8
Numpad 9 / Numpad 0Keypad 9 / 0 (mapped via the left analog-stick X-axis — gearcoleco routes the stick to those two digits)
Numpad *Keypad * (mapped to RetroPad START)
Numpad -Keypad # (mapped to RetroPad SELECT — no `#` on a numpad so we use the adjacent key)

Most games show a "PRESS <number> FOR SKILL LEVEL" screen at boot — that's the keypad in action. Press Numpad 1 for the easiest skill (or whatever number the overlay tells you). All bindings can be remapped via the gear icon in the EmulatorJS toolbar.

Notes

The ColecoVision BIOS (colecovision.rom, ©1982 Coleco) is bundled with this site — the system can't boot without it. The boot screen ("COLECOVISION®") is part of the BIOS and plays every cold reset; this is authentic to the hardware.

Emulator: gearcoleco (GPL-3.0, by Drhelius) under EmulatorJS (GPL-3.0). Both shipped locally from this site; no network calls at runtime.
