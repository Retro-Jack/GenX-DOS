Apple · ][+ (48K, Applesoft BASIC, Disk II)

Apple ][ — Controls

Apple ][js (Will Scullin, MIT) running in your browser as plain JavaScript — no WASM, no Emscripten. The bundled machine is an Apple ][+ with 48 KB RAM, Applesoft BASIC in ROM, and a Disk II controller in slot 6. Each menu entry boots the chosen 5.25" disk image directly; the "Applesoft BASIC" entry boots with no disk inserted, which is what a real ][+ does — it drops to the ] prompt. Click the screen first so the emulator receives keyboard input.

Keyboard

The Apple ][+ keyboard had no lowercase, no arrow keys other than ← →, and a tiny set of modifiers. apple2js maps:

Modern keyApple ][+ key
← →Cursor left/right (the only two arrows on a real ][+)
↑ ↓Synthesised — used for cursor in apps that read them
EscESC (BASIC line edit, app menus)
TabTAB
CtrlCTRL — needed for Ctrl-C (break BASIC), Ctrl-G (bell), Ctrl-Reset
Shift + Ctrl + DeletePower-on reset (cold boot)
F1OPEN APPLE (left ⌘) — fire button 0 in games
F2CLOSED APPLE (right ⌘) — fire button 1 in games

Per-game starting hints

Lemonade StandBoots straight in. Type Y for instructions or N to skip; then number of players and number of days.
Castle Wolfenstein4am crack autoboots. Press Space through the title. Keys: I J K M to move, SPC to fire, R to reload, S to search bodies.
ChoplifterJoystick game — use F1/F2 as the OPEN/CLOSED APPLE buttons or arrows for direction. SPC = fire.
AztecKeys: I J K L compass directions, Q to crouch, U use item, F fire.
Lode RunnerKeys: I J K M = up/left/right/down, U/O dig left/right.
Sammy LightfootJoystick (F1/F2 jump). Press 1 for one-player at the title.
Hard Hat MackKeys: I J K M move, SPC jump.
KaratekaPress Esc to start; D assumes fighting stance. Punches/kicks: J high, K mid, L low (combined with arrows). SPC backs out of stance.
The Oregon TrailType 1/2/3 through the title prompts. Game is keyboard menu-driven throughout.
Applesoft BASICNo disk inserted. The boot ROM hangs at the disk-not-found screen for ~3 seconds then drops to the ] Applesoft prompt. Try 10 PRINT "HELLO": GOTO 10, then RUN.
