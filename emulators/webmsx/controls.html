MSX — Controls

A standard MSX with cart slot, keyboard and a joystick port. WebMSX maps everything to your PC keyboard out of the box; an on-screen menu (the bottom bar) handles disk swap and machine config.

Notes

MSX cartridge games auto-boot. Disk games need to be inserted before the BASIC Ok prompt and reset; multi-disk titles like SD Snatcher use ?ANY= in the URL so you can swap mid-game.

Press Page Up (STOP) twice quickly to break out of a hung BASIC program.

Mapping follows WebMSX (MIT, ppeccin/WebMSX). Hover the bottom HUD for tooltips on every button — the upstream UI has its own help.
