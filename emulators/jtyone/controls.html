Sinclair ZX81 — Controls

A 1981 Sinclair ZX81 with 16K RAM. The membrane keyboard is fully one-key-per-letter, but BASIC keywords are entered as single keypresses depending on cursor mode.

Letter / number keys

The ZX81 keyboard maps directly to your PC keyboard for the four key rows: 1–0, Q–P, A–Enter, Shift–Space.

Mode keys

KeyZX81 key
ShiftSHIFT (combined with letter for keyword / symbol)
EnterNEWLINE
SpaceSPACE / BREAK (use Shift+Space for BREAK)
. (period)FUNCTION-mode "." (also stops a running program when held with Shift)

BASIC keyword entry

The ZX81 enters BASIC keywords as single keys, controlled by the cursor mode shown bottom-left:

CursorBehaviour
K (keyword)The first letter of each keyword types the whole word (P → PRINT, L → LET)
L (letter)Letters type as themselves — used after the keyword on a line
F (function)Press a key after Shift+NewLine to enter a function (e.g. SIN, RND)
G (graphics)Press Shift+9 to enter graphics mode for block characters; Shift+9 again to exit

Game keys

ZX81 games rarely have a standard joystick layout — most use letter keys directly. Common conventions across the bundled titles:

KeyTypical use
5 6 7 8Cursor / movement (Sinclair's "cursor key" overlays — left, down, up, right)
Q A O PUp / Down / Left / Right in many games
0 or SpaceFire / action

In-game instructions appear in the ZX81 Stuff archive page each game came from; check the original loading screen if a key isn't obvious.
