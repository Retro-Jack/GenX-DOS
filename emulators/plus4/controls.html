Commodore · Plus/4

Commodore Plus/4 — Controls

VICE xplus4 (the Plus/4 / C16 / C116 build of the gold-standard Commodore emulator) running in your browser via EmulatorJS. The Plus/4 has 64 KB of RAM and the TED chip (combined video + sound) with a 121-colour palette. Click the canvas first so the emulator receives keyboard input.

When a disk game is loading, the initial read takes around 20-30 seconds — that's not a freeze, it's the real-time 1541 drive emulation reading the disk over the IEC bus at period-authentic speed. You'll hear the drive chunk and stepper-click the whole way through; the game starts when it falls silent. Most of the bundled games are single-file .prgs that load instantly without disk activity.

The Plus/4 has two joystick ports but uses a non-standard 8-pin DIN connector (incompatible with Atari-style sticks). VICE maps Player 1 to port 1; some games look for port 2 instead — use the gear icon's joystick port swap if a game ignores Player 1 input.

Plus/4 keyboard

VICE maps the Plus/4 keyboard "positionally" to your PC keyboard — letters / numbers / punctuation type as themselves. Plus/4-specific keys:

KeyPlus/4 key
L CtrlCTRL
L AltCommodore key (C= key — toggles upper/lower case with SHIFT)
Scroll LockRUN/STOP (remapped from Esc — see note below)
F1 – F4Plus/4 function keys (the four physical F-keys; SHIFTed gives F5/F6/F7 + HELP)
BackspaceINST/DEL
HomeCLR/HOME
InsertSHIFT + INST/DEL (insert)
TabPound symbol (£)
EndESC (Plus/4's actual ESC key — separate from RUN/STOP)

Why Scroll Lock for RUN/STOP? Browsers capture Esc to exit pointer-lock and fullscreen before the libretro keyboard callback can see it, so a real Esc press never reaches VICE. The bundle remaps Scroll Lock via emulators/_shared/genx-vice-softkeys.js — a document-level keydown listener swallows the Scroll Lock press and dispatches a synthetic Escape event (synthetic events don't trigger the browser's Esc-capture since they aren't real user gestures), which then reaches the core as RUN/STOP. Scroll Lock has no other use in this bundle so the remap is free.

Loading from BASIC 3.5

The Plus/4 ships with BASIC 3.5 (a big upgrade over C64's BASIC 2 — full sound/graphics commands). VICE auto-LOADs .prg files and the first program on most disks, but if a game drops you to the READY. prompt, type one of:

CommandEffect
DIRECTORYList the disk directory (Plus/4 BASIC 3.5 shortcut — equivalent to LOAD"$",8 on C64)
DLOAD"name"Load a program from disk (BASIC 3.5 shortcut)
RUNRun the loaded program
