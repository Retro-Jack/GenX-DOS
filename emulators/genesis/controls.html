Notes

Six-button games (Street Fighter II Special Champion Edition, Mortal Kombat 3) use the same A/S/D for the front three buttons; the back row (X/Y/Z) is not mapped in this 3-button build, so combo moves won't all be reachable.

Mapping follows lrusso/Genesis (GPL-2.0, Emscripten port of Notaz's PicoDrive). Verify in upstream README if a control feels off.
