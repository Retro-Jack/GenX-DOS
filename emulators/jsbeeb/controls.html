BBC Micro — Controls

A 1981 Acorn BBC Micro Model B with disc drive. The bundled discs autoboot via ?autoboot; for a clean BASIC > prompt, drop the param.

Keyboard

The BBC keyboard maps directly for letters, numbers and most punctuation. BBC-specific keys:

KeyBBC key
F1 – F10BBC red function keys (also used as in-game action keys by many titles)
F11jsbeeb full-screen

Joystick (analogue)

The BBC's analogue joystick is mapped to mouse motion + a button — point the canvas mouse to deflect, click to fire. Many games also accept Z / X / : / / + Return as the keyboard fallback.

Sideways menu

jsbeeb has a top toolbar with disc swap, model select (Model A/B/Master 128), and a "soft reset" button. Hover the top edge to reveal it.

Notes

If a disc starts at the BASIC > prompt instead of running the game, type *EXEC !BOOT and press Return.

Tape-based titles use CHAIN "" to auto-load the next program from cassette.

Mapping follows jsbeeb v1.12.0 (GPL-3.0+, mattgodbolt/jsbeeb). The full reference (including the on-screen BBC keyboard) is at https://bbc.godbolt.org/.
