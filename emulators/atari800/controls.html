Atari 8-bit — Controls

Atari's 1979–1985 8-bit family. Both 400 (cart-era) and 800XL (disk-era) menus use this same emulator and the same input mapping — only the boot OS differs.

UI

Press F1 to open the atari800 SDL menu. From there you can swap disks (D1: through D8:), toggle BASIC, save/load state, and change machine settings.

Notes

Some 1980s games auto-launch BASIC programs; if a disk drops you to the READY prompt, type RUN "D:<FILENAME>" to start the program. Filenames are visible via DOS at the BASIC prompt.

The 400 menu boots with OS-B (1979 ROM equivalent via AltirraOS-800). The 800XL menu boots with OS-XL (1983 ROM equivalent via AltirraOS-XL). Same keyboard for both.

Mapping follows atari800 v5.2.0 (GPL-2.0+, atari800/atari800). Built to WASM via emscripten with AltirraOS baked in. The full key reference (including international keyboards) lives in the upstream DOC/USAGE.
