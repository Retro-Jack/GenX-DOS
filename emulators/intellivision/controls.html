Intellivision — Controls

Self-hosted jzIntv (Joe Zbiciak's gold-standard Intellivision emulator) compiled to WebAssembly, mirrored from mholzinger/intellivision-overlay-editor. Launched 1979 by Mattel Electronics — the first 16-bit home console (CP1610 CPU), the first console with a controller that had a 16-direction disc, a 12-key keypad, and 4 action buttons. Games shipped with cardboard keypad overlays. Click the canvas first so the emulator receives keyboard input.

Disc — directional input

The Intellivision controller's flat disc was the first home-console pad with 16-way input (real PC-keyboard arrow keys only cover 8 — jzIntv synthesizes the in-between angles when you hold two arrows). Used for movement, aiming, steering, and disc-throwing in TRON.

Action buttons — top, bottom-left, bottom-right

Three positions, four buttons total — the two bottom buttons are wired identically so most games treat them as one "fire" button. Each game's overlay tells you what each does.

Keypad — overlay buttons

12 numeric keys (1-9, CLR, 0, ENT). Period-correct shape is the PC numpad — same 3×4 grid as the original cardboard overlay.

Player 2 controller

Right-hand modifier keys map to Controller 2's action buttons. The keypad isn't dual-mapped here — most 2-player Intellivision games use the disc + action buttons for both players (Sea Battle, Auto Racing, Hockey, etc.).

Notes

The Intellivision BIOS files (exec.bin + grom.bin, ©1979 Mattel Electronics) are bundled with this site — the system can't boot without them. The boot screen briefly shows "Mattel Electronics Presents" before the title launches.
