Notes

Player 2 isn't mapped in this build — JSNES focuses on a single keyboard controller.

The Zapper light gun (Duck Hunt etc.) isn't supported by JSNES; those games will run but can't register hits.

The browser may suspend audio until you click the canvas — this is the standard Web Audio autoplay policy, not a bug.

Mapping follows JSNES v2.1.0 (Apache-2.0, bfirsh/jsnes). Verify in upstream README if a control feels off.
