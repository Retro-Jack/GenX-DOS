Commodore · 128

Commodore 128 — Controls

Self-hosted VICE x128 (the gold-standard Commodore 128 emulator) running in your browser via EmulatorJS, configured for the C128 in native mode (BASIC 7.0, 128 KB RAM, dual VIC-IIe + VDC video chips, 8502 + Z80 CPUs). The 128 famously had three personalities: this bundle skips C64 compatibility mode (we have a dedicated C64 bundle for that) and skips CP/M mode, focusing on the 128's distinctive native features. Click the canvas first so the emulator receives keyboard input.

Each game's games.json entry sets the right video chip up-front — VIC-IIe (40 columns, same as the C64) for action games and VDC (80 columns) for the Infocom text adventures and Star Fleet I that need the wider display. You can manually swap chips at any time by pressing F7 (the C128's "40/80 DISPLAY" toggle key).

On first load, click the canvas once — the C128 KERNAL takes ~2 seconds to boot (RAM test, dual-video init, BASIC 7.0 setup) and the browser's audio context can't start without a user gesture, so the boot stays frozen on whatever frame it reached until you click. After that single click, boot proceeds normally and finishes at the READY. prompt.

Every bundled game is a single-disk title — no mid-session disk swaps required. (Trinity and Beyond Zork were on the shortlist but their C128 80-column releases ship as dual-sided 1571 D71 images that prompt for disk B mid-game, so they were swapped for the single-disk Knight's Quest and Wumpus 2.0 instead.)

Joystick is mapped to the numeric keypad via VICE's vice_joyport_type='1' setting so it doesn't conflict with the keyboard. NumLock needs to be on for the numpad to fire — a banner reminds you if it isn't.

Why Scroll Lock for RUN/STOP? Browsers capture Esc to exit pointer-lock and fullscreen before the libretro keyboard callback can see it, so a real Esc press never reaches VICE. The bundle remaps Scroll Lock via emulators/_shared/genx-vice-softkeys.js — a document-level keydown listener swallows the Scroll Lock press and dispatches a synthetic Escape event (synthetic events don't trigger the browser's Esc-capture since they aren't real user gestures), which then reaches the core as RUN/STOP. Scroll Lock has no other use in this bundle so the remap is free.

Why the C128 library is small

The C128 sold ~5 million units but its native mode (BASIC 7.0, 80-column VDC, 2 MHz fast mode) was used by only ~100-200 commercial titles total — most developers targeted the C64 install base and shipped C64-compatible code that ran on the 128 via GO64. The native-mode library is dominated by productivity software (Paperback Writer 128, GEOS 128, Big Blue Reader) and the Infocom 80-column text adventures bundled here. Of the action games, only a handful (The Last V8, Rocky Horror Show, Thai Boxing, Kikstart 2) used the 128's native mode meaningfully.

Emulator: VICE x128 (GPL-2.0, libretro core) under EmulatorJS (GPL-3.0). Both shipped locally from this site; no network calls at runtime.
