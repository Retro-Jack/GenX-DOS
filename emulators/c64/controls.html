Commodore 64 — Controls

Each C64 game opens in an iframe hosted by the Internet Archive — they serve their own emulator (VICE x64 under emularity) with the disc image already mounted. Click the canvas first so the emulator receives keyboard input.

Joystick (port 2 — Player 1)

KeyAction
↑ ↓ ← →Stick direction
L CtrlFire button

Most C64 games use port 2 by default. If a game ignores Player 1 input, look in archive.org's emulator toolbar (bottom of the iframe) for a controller-swap option.

C64 keyboard

VICE maps the C64 keyboard "positionally" to your PC keyboard — letters / numbers / punctuation type as themselves. C64-specific keys:

Loading from BASIC

VICE auto-LOADs the first program on most disks, but if a game drops you to the READY. prompt, type one of:

CommandEffect
LOAD"$",8List the directory of disk drive 8
LOAD"*",8,1Load + run the first program on the disk
RUNRun the loaded program
