Tandy · Color Computer (CoCo)

Tandy CoCo — Controls

A Tandy/Radio Shack Color Computer 1/2 with Color BASIC. The bundled menu boots cartridges; the Color BASIC OK prompt is one menu entry away.

Keyboard

The CoCo's 53-key keyboard maps directly to your PC keyboard for letters, numbers and most punctuation. CoCo-specific keys:

KeyCoCo key
EnterENTER
Backspace← (BACK / left-arrow, used to delete)
↑ ↓ ← →CoCo arrow keys (also used for in-game movement)
Shift + 0BREAK (interrupt running program)
F1XRoar runtime menu (machine + cart + tape config)

Joysticks

The CoCo has two analogue joystick ports. XRoar emulates both via the keyboard:

KeyAction
↑ ↓ ← →Right joystick (most cartridge games)
L AltRight joystick fire button
W A S DLeft joystick
L CtrlLeft joystick fire button

System / cassette

KeyAction
F1Open XRoar menu (drag/drop a tape, change machine, reset)
F12Hard reset
Shift + F12Soft reset (warm boot)

Notes

Cart games typically auto-launch from the cart slot — there's no CLOAD step. Tape games (rare in this build) need CLOAD + RUN from BASIC.

Dungeons of Daggorath uses keys + Shift-modifiers heavily — see its bundled docs at colorcomputerarchive.com for the full key chart.

Mapping follows XRoar v1.10 (GPL-3.0+, Ciaran Anscomb). The full doc/xroar.html in the upstream tree has every key and shortcut.
